Assistant capabilities

  • Semantic retrieval: Uses local ChromaDB and sentence embeddings for grounded answers.
  • Structured lookups: Pulls player stats, rules, and records from curated local knowledge.
  • Comparison flows: Handles player-versus-player and season-vs-season questions naturally.

Experience principles

  • Hierarchy first: Clear headline, supporting copy, and visible feature chips.
  • Low visual noise: Strong contrast, limited accent colors, and controlled glass effects.
  • Responsive by default: Layout collapses cleanly for smaller screens.

System integrations

  • FastAPI + Gradio: Local app shell with an embedded assistant interface.
  • Gemini and Ollama: Optional LLM paths with automatic model fallback.
  • Local-only operation: Designed to run without cloud services for core functionality.
Built with Gradio, FastAPI, ChromaDB, and LangChain.
